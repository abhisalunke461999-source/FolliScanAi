# Folliscan 🧬

**Folliscan** is an AI-driven web application designed to help users analyze scalp health and receive personalized nutritional guidance to combat hair fall.

## 🚀 Live Site
Check out the project here: [https://folliscan.in](https://folliscan.in)

## 🛠️ Features (In Development)
* **Scalp Health Analysis:** AI-powered image recognition to detect scalp conditions.
* **Nutritional Guidance:** Personalized diet plans focusing on protein-rich foods and vitamins.
* **Progress Tracking:** Monitor hair density and scalp improvement over time.

## 📂 Project Structure
* `index.html` - The main landing page and entry point.
* `CNAME` - Configures the custom domain for GitHub Pages.
* `README.md` - Project documentation (this file).

## 🔧 Setup & Deployment
This site is hosted using **GitHub Pages**. To deploy changes:
1. Push your code to the `main` branch.
2. Ensure `index.html` is in the root directory.
3. GitHub Actions will automatically build and deploy the site to `folliscan.in`.
